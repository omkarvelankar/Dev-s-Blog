First Website Using VScode





We will be creating our first website using VS Code editor. Well, we can use notepad but with VS Code it becomes easy to handle all the related files in one place.


Download VS Code from the Visual Studio Code website and install it.


Create a folder at your desired location and open it in VS Code. Create New File and name it home.html and also index.html will be a good name too. Nowadays GitHub is very popular. And if we want to host our website on GitHub. We should name our main page index.html


Shortcut in VS code shift+! (remember to name the file .html to make this shortcut work) will produce this code 

<!DOCTYPE html>

<html lang="en">

<head>

<meta charset="UTF-8">

<meta name="viewport" content="width=device-width, initial-scale=1.0">

<meta http-equiv="X-UA-Compatible" content="ie=edge">

<title>Document</title>

</head >

<body>

</body>

</html>

Edit the title section <title> Your desired title </title> to name the document as you like.

In my case its Dev's Blog
