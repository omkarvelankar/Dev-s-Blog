Use of Links

Links are useful in navigating to other desired pages. By default, a link will appear like this (in all browsers): An unvisited link is underlined and blue. A visited link is underlined and purple. An active link is underlined and red. If you are following steps from previous articles you will be able to link like I am going to do now. Go to VS Code and open up the project.

Anchor Tag

Links are created using anchor tag Attributes:- href is used to provide an address to the link

< href="address"> Text that you want provide link to > Let's create three HTML files
      about.html
      portfolio.html
      contact.html

Add the following code as per below in between body tags in the index.html file

    <div>
    <href="" >Dev's Blog  </a>
    <href="" >About  </a>
    <href="" >Portfolio  </a>
    <href="" >Contact  </a>
    </div>

As all the files are located in the same folder we are going to use VS Code shortcut to provide the address to href Click in between " " marks in front of href and press the forward slash ( / ) button. And you will be able to see available files which you can add as a addresses

Select the addresses. Click Save. Go to the browser and you will be able to see the links.

Click on the links and you will be navigated to your respective pages.
