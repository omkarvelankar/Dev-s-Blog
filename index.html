What is a Website?

What is a website? have you ever wondered? Many of you might even know what is a web site and think what kind of silly question is this?

Well, silly might it seems. But is very important to know.

A website is made of web pages. It needs a domain name and hosting.

Types of Websites.

Static:

A simple website where data does not needs to be updated frequently. Like Informational websites about businesses, organizations, etc

Dynamic:

Dynamics websites are the websites where data is loaded from the backend. It changes according to user permissions .Websites like Facebook, LinkedIn, etc

Points covered:

  • What is a wireframe?
  • What wireframes should include?
  • What wireframes should not have?

What is a wireframe?

Before manufacturing a car or constructing a building we need to build its design on paper. Likewise before making a website we need to design it on paper.

Wireframing is the first initial step in designing a website. It is the blueprint of the design of the webpage. One needs a basic understanding of wireframing for creating websites. It is like creating a skeleton.

What wireframes should include?

  • The type of content will be displayed on the web page (not the content itself)
  • Structure and Layout of the Webpage.
  • User Stories and flow.

What it should not include?

  • Colors.
  • Texts.
  • Images.

HTML Tag <html>

Every webpage will start with <html> and end with </html>

Ex. 
    <html>
      html Tags ..... 
    </html>

Head tag - <head> :

The head tag <head> is a container for all the head elements.
The head tag <head> can include a title for the document, scripts, styles, meta information, and more.


Ex. 
            

        <head>

            <meta> charset="UTF-8"> 
 

            <meta http-equiv="X-UA-Compatible" content="IE=edge"> 
 

            <meta name="viewport" content="width=device-width, initial-scale=1.0"> 
 

            <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.1.3/dist/css/bootstrap.min.css" rel="stylesheet"> 
 

            <script src="https://cdn.jsdelivr.net/npm/bootstrap@5.1.3/dist/js/bootstrap.bundle.min.js"></script> 


            <link href="D:\website\main.css" rel="stylesheet" > 
 

            <title>Document</title> 
 

        </head>

Title Tag <title> :

Title tag <title> needs to be placed within the head tag. It defines a title in the browser toolbar
Provides a title for the page when it is added to favorites
Title Tag <title> displays a title for the page in search engine results


Ex. 
            </head>
            <title> Dev's Blog </title>
            </head>

Header Tag <h1>, <h2>, <h3>, <h4>, <h5>, <h6> :

The header tags <h1> to <h6> tags are used to define HTML headings. <h1> defines the most important heading. <h6> defines the least important heading.


Ex. 

        <h1> Topics <h1>

            <h2>HTML<h2>
           
            <h2>CSS<h2>
           
            <h2>Javascript<h2>

Body Tag <body>

Body tag <body> contains all the contents of an HTML document, such as text, hyperlinks, images, tables, lists, etc.

Within <body> tag other tags are used like <div></div>, <p></p>, <img>, <form><form>, etc


Ex. 
           <body>
            
            <div>

            <p>
            Dev's Blog is a series of articles on web development. This is an effort by the writer to help learn about web development technologies.
            </p>
           
            </div>
           
           </body>

Div Tag

The <div> tag defines a division or a section in an HTML document.

Ex. 

    <div>

    <p>
    Dev's Blog is a series of articles on web development. This is an effort by the writer to help learn about web development technologies.
    </p>
   
    </div>

Paragraph Tag <p>

The <p> tag defines a paragraph.
These are the basic tags used for text.

Ex. 
            <p>
            Dev's Blog is a series of articles on web development. This is an effort by the writer to help learn about web development technologies.
            </p>
