Basic HTML Tags





Points covered:

  • HTML tag - <html>
  • head tag - <head>
  • body tag - <body>
  • header tags - <h1>,<h2>,<h3>,<h4>,<h5>,<h6>
  • div tag - <div>
  • paragraph tag - <p>
  • Example Structure

HTML Tag <html>

Every webpage will start with <html> and end with </html>

Ex. 
<html>
   html Tags ..... 
</html>

Head tag - <head> :

The head tag <head> is a container for all the head elements.
The head tag <head> can include a title for the document, scripts, styles, meta information, and more.


Ex. 
    

<head>

    <meta> charset="UTF-8"> 
 

    <meta http-equiv="X-UA-Compatible" content="IE=edge"> 
 

    <meta name="viewport" content="width=device-width, initial-scale=1.0"> 
 

    <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.1.3/dist/css/bootstrap.min.css" rel="stylesheet"> 
 

    <script src="https://cdn.jsdelivr.net/npm/bootstrap@5.1.3/dist/js/bootstrap.bundle.min.js"></script> 


    <link href="D:\website\main.css" rel="stylesheet" > 
 

    <title>Document</title> 
 

</head>

Title Tag <title> :

Title tag <title> needs to be placed within the head tag. It defines a title in the browser toolbar
Provides a title for the page when it is added to favorites
Title Tag <title> displays a title for the page in search engine results


Ex. 
</head>
<title> Dev's Blog </title>
</head>

Header Tag <h1>, <h2>, <h3>, <h4>, <h5>, <h6> :

The header tags <h1> to <h6> tags are used to define HTML headings. <h1> defines the most important heading. <h6> defines the least important heading.


Ex. 

<h1> Topics <h1>

<h2>HTML<h2>
   
<h2>CSS<h2>
   
<h2>Javascript<h2>

Body Tag <body>

Body tag <body> contains all the contents of an HTML document, such as text, hyperlinks, images, tables, lists, etc.

Within <body> tag other tags are used like <div></div>, <p></p>, <img>, <form><form>, etc


Ex. 
   <body>
    
    <div>

    <p>
    Dev's Blog is a series of articles on web development. This is an effort by the writer to help learn about web development technologies.
    </p>
   
    </div>
   
   </body>

Div Tag

The <div> tag defines a division or a section in an HTML document.

Ex. 

<div>

<p>
Dev's Blog is a series of articles on web development. This is an effort by the writer to help learn about web development technologies.
</p>

</div>

Paragraph Tag <p>

The <p> tag defines a paragraph.
These are the basic tags used for text.

Ex. 
    <p>
    Dev's Blog is a series of articles on web development. This is an effort by the writer to help learn about web development technologies.
    </p>
