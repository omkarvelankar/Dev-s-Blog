What is a Website?

What is a website? have you ever wondered? Many of you might even know what is a web site and think what kind of silly question is this?

Well, silly might it seems. But is very important to know.

A website is made of web pages. It needs a domain name and hosting.

Types of Websites.

Static:

A simple website where data does not needs to be updated frequently. Like Informational websites about businesses, organizations, etc

Dynamic:

Dynamics websites are the websites where data is loaded from the backend. It changes according to user permissions .Websites like Facebook, LinkedIn, etc. We will learn more about it after learning how to create simple HTML static website.

Things needed to develop a Static website: A simple static website consisting of web pages can be made using a simple word editor like notepad(my personal favorite).
